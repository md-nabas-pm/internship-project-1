# Icons Folder

Store all icons used in the project.

## What to Add

- Navigation icons
- Social media icons
- UI icons
- Menu icons
- Buttons/icons

## Recommended Format

```txt
SVG (Preferred)
PNG (Fallback)
```

SVG is preferred because:
- Better quality
- Scalable
- Lightweight

## Naming Convention

```txt
menu.svg
search.svg
close.svg
facebook.svg
instagram.svg
```

Avoid:

```txt
icon1.svg
newiconfinal.svg
```

## Downloading Icons from Figma

1. Select icon in Figma.
2. Go to **Export** section.
3. Select:

```txt
SVG
```

4. Click **Export**.
5. Save to:

```txt
assets/icons/
```

## Usage Example

```html
<img src="assets/icons/menu.svg" alt="Menu Icon">
```

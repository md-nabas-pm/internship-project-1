
# Images Folder

Store all project images here.

## What to Add

- Logos
- Hero banners
- Background images
- UI illustrations
- Product images

## Supported Formats

```txt
PNG
JPG / JPEG
WEBP
SVG (if required)
```

## Naming Convention

Use clean file names:

```txt
hero-banner.png
company-logo.png
about-image.jpg
```

Avoid:

```txt
IMG_1234.png
final-new-latest.png
```

## Downloading Images from Figma

1. Open Figma design.
2. Select the image or frame.
3. Open right sidebar → **Export** section.
4. Click **+ Export**.
5. Choose format:

```txt
PNG → General UI Images
JPG → Photos
SVG → Vector Graphics
WEBP → Optimized Images
```

6. Click **Export**.
7. Save inside:

```txt
assets/images/
```

## Usage Example

```html
<img src="assets/images/logo.png" alt="Logo">
```
